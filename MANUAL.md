## การติดตั้งระบบ

### ขั้นตอนที่ 1: ติดตั้ง Dependencies
```bash
npm install
```

### ขั้นตอนที่ 2: ตั้งค่าฐานข้อมูล

**วิธีที่ 1: ใช้ Batch Script (แนะนำสำหรับ Windows)**
```bash
setup-database.bat
```

**วิธีที่ 2: ทำด้วยตนเอง**

1. เข้า MySQL:
```bash
mysql -u root -p
```

2. รัน schema:
```sql
source database/schema.sql
```

3. Initialize users:
```bash
npm run init-db
```

### ขั้นตอนที่ 3: รันเซิร์ฟเวอร์

**วิธีที่ 1: ใช้ Batch Script**
```bash
start-server.bat
```

**วิธีที่ 2: ใช้ npm**
```bash
npm run dev
```

เปิดเบราว์เซอร์ที่: **http://localhost:3000**

## การใช้งานสำหรับ Admin

### 1. เข้าสู่ระบบ
- Username: `admin`
- Password: `admin123`

⚠️ **ควรเปลี่ยนรหัสผ่านทันทีหลังเข้าใช้งานครั้งแรก**

### 2. หน้าหลัก (Dashboard)
- แสดงสถิติภาพรวมของระบบ
- จำนวนรายการทั้งหมด
- ชั่วโมงการอบรมรวม
- จำนวนผู้ใช้งาน
- รายการล่าสุด

### 3. จัดการข้อมูลการอบรม

#### เพิ่มรายการใหม่
1. คลิกเมนู "ประวัติการอบรม"
2. คลิกปุ่ม "+ เพิ่มรายการใหม่"
3. กรอกข้อมูล:
   - **หัวข้อ*** (บังคับ)
   - **ประเภท*** (การอบรม/การประชุม/ฯลฯ)
   - **รูปแบบ*** (ออนไลน์/ออนไซต์)
   - **วันที่เริ่ม - วันที่สิ้นสุด*** (บังคับ)
   - **จำนวนชั่วโมง*** (บังคับ)
   - หน่วยงาน
   - สถานที่
   - รายละเอียด
   - เลขที่ใบรับรอง
4. คลิก "บันทึก"

#### แก้ไขรายการ
1. คลิกปุ่ม "แก้ไข" ในรายการที่ต้องการ
2. แก้ไขข้อมูล
3. คลิก "บันทึก"

#### ลบรายการ
1. คลิกปุ่ม "ลบ" ในรายการที่ต้องการ
2. ยืนยันการลบ

#### ค้นหาและกรองข้อมูล
- **ช่องค้นหา**: พิมพ์คำค้นหาในหัวข้อหรือรายละเอียด
- **กรองตามประเภท**: เลือกประเภทที่ต้องการ
- **กรองตามรูปแบบ**: เลือกรูปแบบที่ต้องการ

### 4. จัดการผู้ใช้งาน

#### เพิ่มผู้ใช้ใหม่
1. คลิกเมนู "จัดการผู้ใช้"
2. คลิกปุ่ม "+ เพิ่มผู้ใช้ใหม่"
3. กรอกข้อมูล:
   - **รหัสพนักงาร*** (บังคับ)
   - **ชื่อผู้ใช้*** (บังคับ)
   - **ชื่อ-นามสกุล*** (บังคับ)
   - **อีเมล*** (บังคับ)
   - **รหัสผ่าน*** (บังคับ)
   - ตำแหน่ง
   - แผนก
   - **สิทธิ์*** (ผู้ใช้ทั่วไป/ผู้ดูแลระบบ)
4. คลิก "บันทึก"

#### แก้ไขข้อมูลผู้ใช้
1. คลิกปุ่ม "แก้ไข" ในรายการผู้ใช้
2. แก้ไขข้อมูล (ไม่ต้องใส่รหัสผ่านถ้าไม่ต้องการเปลี่ยน)
3. คลิก "บันทึก"

#### ลบผู้ใช้
1. คลิกปุ่ม "ลบ" ในรายการผู้ใช้
2. ยืนยันการลบ

## การใช้งานสำหรับ User

### 1. เข้าสู่ระบบ
- ใช้ Username และ Password ที่ Admin สร้างให้

### 2. ดูประวัติการอบรมของตนเอง
- หน้าหลักจะแสดงสถิติส่วนตัว
- เมนู "ประวัติการอบรม" จะแสดงเฉพาะข้อมูลของตนเอง

### 3. เพิ่มข้อมูลการอบรม
- สามารถเพิ่มข้อมูลการอบรมของตนเองได้
- ขั้นตอนเหมือนกับ Admin

### 4. แก้ไข/ลบข้อมูล
- สามารถแก้ไขหรือลบเฉพาะข้อมูลของตนเองเท่านั้น

### 5. เปลี่ยนรหัสผ่าน
1. คลิกที่ชื่อผู้ใช้มุมขวาบน
2. เลือก "เปลี่ยนรหัสผ่าน"
3. กรอกรหัสผ่านเดิมและรหัสผ่านใหม่
4. คลิก "บันทึก"

## คำถามที่พบบ่อย

### Q: ลืมรหัสผ่านทำอย่างไร?
**A:** ติดต่อ Admin เพื่อขอรีเซ็ตรหัสผ่าน

### Q: สามารถเพิ่มข้อมูลย้อนหลังได้หรือไม่?
**A:** ได้ สามารถเลือกวันที่ย้อนหลังได้

### Q: ข้อมูลจะถูกลบถาวรหรือไม่?
**A:** ใช่ เมื่อลบแล้วจะไม่สามารถกู้คืนได้

### Q: สามารถส่งออกข้อมูลเป็น Excel ได้หรือไม่?
**A:** ฟีเจอร์นี้จะพัฒนาในเวอร์ชันถัดไป

### Q: รองรับการแนบไฟล์หรือไม่?
**A:** ฟีเจอร์นี้จะพัฒนาในเวอร์ชันถัดไป

### Q: ระบบรองรับมือถือหรือไม่?
**A:** ใช่ ระบบออกแบบให้ใช้งานบนมือถือได้
